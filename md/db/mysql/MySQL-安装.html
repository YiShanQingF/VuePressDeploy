# MySQL 安装

本文档记录在Windows下，使用 ZIP Archive 安装 MySQL-8.0.34-winx64。

# 官网下载

  • ZIP Archive下载

最新ZIP Archive下载地址：https://dev.mysql.com/downloads/mysql/ (opens new window)

历史ZIP Archive版本下载地址：https://downloads.mysql.com/archives/community/ (opens new window)

  • MSI Installer下载

最新MSI Installer下载地址：https://dev.mysql.com/downloads/windows/installer/ (opens new window)

最新MSI Installer下载地址：https://downloads.mysql.com/archives/installer/ (opens new window)

# 准备操作

# 解压文件

  1. 将mysql-8.0.34-winx64.zip文件解压到安装路径，例：D:\software\Mysql\mysql-8.0.34-winx64
  2. 创建数据存放路径，例：D:\software\Mysql\data

# 环境变量

  1. 新增环境变量mysql_home，设置路径D:\software\Mysql\mysql-8.0.34-winx64
  2. 新增path环境变量，设置路径%mysql_home%\bin

# 配置文件

  1. 在mysql跟路径创建配置文件my.ini，修改basedirmysql的安装目录和datadirmysql数据库的数据的存放目录
  2. 注意路径使用**\**
[mysqld]
# 设置3306端口
port=3306
# 设置mysql的安装目录
basedir=D:\\software\\Mysql\\mysql-8.0.34-winx64
# 设置mysql数据库的数据的存放目录
datadir=D:\\software\\Mysql\\data
# 允许最大连接数
max_connections=200
# 允许连接失败的次数。这是为了防止有人从该主机试图攻击数据库系统
max_connect_errors=10
# 服务端使用的字符集默认为 utf8mb4
character-set-server=utf8mb4
# 创建新表时将使用的默认存储引擎
default-storage-engine=INNODB
# 默认使用“mysql_native_password”插件认证
default_authentication_plugin=mysql_native_password
[mysql]
# 设置mysql客户端默认字符集
default-character-set=utf8mb4
[client]
# 设置mysql客户端连接服务端时默认使用的端口
port=3306

# 安装、启动MySQL

以管理员身份运行 CMD，且定位到安装路径的bin目录下，记录输出日志的初始化密码。

  1. 添加mysqld到服务：mysqld --install mysql
  2. mysql初始化：mysqld --initialize --console
  3. 启动mysql：net start mysql

# 修改初始化密码

  1. 登录mysql客户端

mysql -h 主机名 -u 用户名 -p

-h : 指定客户端所要登录的 MySQL 主机名, 登录本机(localhost 或 127.0.0.1)该参数可以省略;

-u : 登录的用户名;

-p : 告诉服务器将会使用一个密码来登录, 如果所要登录的用户名密码为空, 可以忽略此选项。

mysql -u root -p 本机使用该命令即可

  1. 修改密码

ALTER USER 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY 'root';

  1. 加密方式

ALTER USER 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY 'root';

ALTER USER 'root'@'localhost' IDENTIFIED WITH caching_sha2_password BY 'root';

caching_sha2_password 表示使用8.0新版本加密方式。

select user,host,plugin from mysql.user; 查询用户的加密方式

# 开通远程连接

  1. 查看MySQL当前访问远程访问权限

use mysql;

select User,authentication_string,Host from user;

  1. 开启远程访问

update user set host='%' where user='root';

# 数据库卸载

  1. 停止mysql 服务：net stop mysql
  2. 卸载mysql 服务：mysqld --remove mysql
  3. 删除产生的data文件夹
